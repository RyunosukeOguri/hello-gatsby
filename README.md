# Start

```shell
$ git clone
$ cd hello-gatsby
$ yarn install
$ yarn develop
```

# Gatsby の初め方

gatsby cli をインストールします

```shell
$ npm install -g gatsby-cli
```

※CLI を使わない場合は下記で gatsby をインストールできます。

```shell
$ npm init gatsby
```

```
$ gatsby new hello-gatsby
$ cd hello-gatsby
$ gatsby develop or yarn develop
```
